# Veer Auth Service

Multi-module Maven project for Veer authentication service.

## Modules

### veer-auth-api
REST API controllers and interfaces for external communication.

### veer-auth-model
Domain models, DTOs, and data structures.

### veer-auth-service
Main service implementation with business logic.

## Building

```bash
mvn clean install
```

## Running

```bash
cd veer-auth-service
mvn spring-boot:run
```

## Requirements

- Java 17+
- Maven 3.6+
